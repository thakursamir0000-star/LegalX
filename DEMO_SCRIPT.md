# LegalX Knowledge Centre - Demo Video Script

## Recording Setup
- **Duration:** 5-7 minutes
- **Resolution:** 1920x1080 (1080p) or 1280x720 (720p minimum)
- **Tools:** OBS Studio, ScreenFlow (Mac), or built-in screen recorder
- **Audio:** Clear mic, no background noise
- **Pace:** Slow enough to read; fast enough to stay engaged

---

## Scene 6: Chat Assistant - RAG Q&A (4:00-5:30)

### Visual
- Click on "Chat" tab
- Show chat interface with:
  - Message history area
  - Input text box
  - Suggested questions buttons
  - Loading spinner (animated)

### Narration
"The Chat Assistant is powered by Retrieval-Augmented Generation (RAG). Instead of just using a general AI model, our system:
1. Searches our vector database for the most relevant sections of the law
2. Passes those sections as context to Gemini
3. Generates accurate, sourced answers

This ensures answers are grounded in the actual legal text and can cite sources."

### Actions
1. Click Chat tab
2. Show the suggested questions
3. Click on "What is the punishment under POCSO?" 
4. Wait for response (2-3 sec)
5. Read the answer - notice it's specific and sourced
6. Point out source citation
7. Ask follow-up question: "Who can file a complaint?"
8. Show that chat history is maintained
9. Ask another question to demonstrate multi-turn conversation

### Timing Notes
- Each question takes 2-3 seconds to respond
- Show the loading animation while waiting
- Read the response to audience

## Tips for Recording

1. **Practice first:** Do a dry run before recording
2. **Speak clearly:** Enunciate, not too fast
3. **Point to UI elements:** Use cursor to highlight what you're talking about
4. **Pause between sections:** Give viewers time to read text
5. **Load times:** Acknowledge waiting for API responses
6. **Mistakes:** It's okay to re-record sections
7. **Editing:** Add title card at start, intro text, close-ups where helpful
8. **Audio:** Add background music (royalty-free) at low volume
9. **Final check:** Watch the full demo before submitting

- Format: MP4
- Resolution: 1920x1080 or 1280x720
- Bitrate: 5000-8000 kbps
- Duration: 5-7 minutes
- File size: Aim for ~300-500 MB for quick upload

- YouTube: Public or Unlisted
